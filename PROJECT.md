# Project Summary

The Enterprise Real-Time Fraud Detection & Payment Intelligence Platform is a complete data engineering portfolio project focused on production-style fraud analytics.

## Goals

- Demonstrate end-to-end streaming and batch data engineering.
- Show secure, credential-free open-source configuration practices.
- Provide a recruiter-friendly example of Airflow, Kafka, Spark, Databricks, dbt, Docker, CI, and Streamlit working together.
- Keep business logic stable while making the repository publishable.

## Non-Goals

- This is not a production fraud decisioning service.
- This repository does not include live cloud credentials.
- CI validates build quality and syntax; it does not run cloud jobs.

## Primary Audiences

- Recruiters and hiring managers reviewing data engineering depth.
- Engineers studying reference architecture patterns.
- Maintainers preparing the project for public GitHub release.
