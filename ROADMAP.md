# Roadmap

The core project is complete. Future work should focus on polish, reproducibility, and operational maturity rather than new business features.

## Release Readiness

- Add a formal open-source license.
- Add masked dashboard screenshots under `docs/images/`.
- Add sample non-sensitive synthetic data for a lightweight public demo.

## Engineering Improvements

- Add infrastructure-as-code examples for Databricks jobs and cloud resources.
- Add lineage and observability with OpenLineage-compatible metadata.
- Add data quality checks with Great Expectations, Soda, or dbt constraints.

## Deployment Improvements

- Add a Streamlit Community Cloud deployment guide with screenshots.
- Add a Docker Compose profile for optional dashboard runtime.
- Add documented local dbt profiles for DuckDB and Databricks.
